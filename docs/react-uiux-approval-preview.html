React 반영 전 컨펌용 시안

설문 화면을 결정 흐름으로 바꾸는 안

이 파일은 실제 React 코드에 반영하기 전, 사용자가 방향을 승인할 수 있도록 만든 정적 미리보기입니다. 핵심은 질문지를 줄이고, 결정 카드와 계획 미리보기로 판단을 쉽게 만드는 것입니다.

제안 요약

바꾸는 이유

  • 현재 화면은 설문 작성 느낌이 강함
  • 선택 결과가 계획에 어떻게 반영되는지 약함
  • 기본 선택값 때문에 대충 넘기기 쉬움

새 방향

결정 카드 상태 체크 칩 선택 계획 미리보기

반영 범위

  • HomeView CTA 문구 조정
  • DecisionView 구조 개편
  • PlanPreviewView 편집감 강화
  • Store 구조는 최대한 유지
